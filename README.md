## 💼 Sobre mí
Soy un desarrollador enfocado en la construcción de proyectos web funcionales, con interés en el manejo eficiente de datos y la lógica detrás de cada sistema.

Trabajo con tecnologías como HTML, CSS, JavaScript y Python, complementando mis soluciones con bases de datos como MySQL y MongoDB.

Me apoyo en herramientas como Git y GitHub para el control de versiones, utilizando la terminal como parte de mi flujo de trabajo diario y Visual Studio Code como entorno principal.

Además, desarrollo habilidades creativas en edición de video (CapCut, DaVinci Resolve), diseño (Canva) y producción musical (FL Studio), integrando lo técnico con lo visual y sonoro.
